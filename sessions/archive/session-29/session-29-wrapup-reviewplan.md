## Execution Protocol

### Phase 1: Team Syntheses (5 parallel teams)

Each team is a single agent (the designated writer) that:
1. Reads ALL collab papers assigned to their team
2. Identifies convergent themes across team members
3. Identifies divergent assessments with reasoning from each side
4. Produces a priority-ordered computation list for Session 30
5. Writes `session-29-team-{X}-synthesis.md`

**Model**: Sonnet for all 5 writers (cost-efficient, these are synthesis not original physics)

### Phase 2: Report

After all 5 team syntheses complete:
- Report completion status
- List output files
- Summarize top-3 convergent themes and top-3 divergent assessments across all teams
