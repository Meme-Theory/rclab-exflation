## Reading Levels

**Level 0 — Triage**: Read the **Quick Reference** table in `index.md`. Match your task keywords to paper numbers. If nothing matches, this folder isn't relevant — move on.

**Level 1 — Targeted**: Go to the **Paper Entries** for the papers you identified. Read the Summary, Key Results, and Key Equations table. This is enough for most tasks.

**Level 2 — Deep**: Open the actual `.md` paper file. Use equation labels from the index to jump to the right section. Check **Dependencies** to know what upstream papers to also read.

## Rules

1. Start at Level 0. Only escalate when you need more detail.
2. Use **tags** on paper entries to find related papers.
3. Read upstream dependencies before downstream papers.
4. If a later paper extends or corrects an earlier one, the later paper takes precedence.
5. **Relevance ratings** (CRITICAL / HIGH / MEDIUM / LOW) are relative to the current project scope.
