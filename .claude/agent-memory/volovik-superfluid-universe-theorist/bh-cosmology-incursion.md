## BH Cosmology Incursion Findings (2026-03-13)

### Central Result (REVISED by Section I.10 Addendum)
The ORIGINAL result (Section I.5): Lambda* = 0 exactly from symmetric Gibbs-Duhem matching.
The CORRECTED result (Section I.10): The symmetric matching assumed bidirectional exchange, but the BH horizon is a one-way membrane. An accretion source term J(t) = (dM_BH/dt * c^2) / V_child breaks the symmetry. The fixed point is displaced to Lambda* = J*/(3*H*) != 0.

However, numerical estimate for the self-similar recursive case: J/Lambda_obs ~ 4 x 10^{-19}. The faucet correction is 19 orders of magnitude below the observed CC. Lambda* ~ 0 remains a good approximation. The flaw is real but perturbatively small.
