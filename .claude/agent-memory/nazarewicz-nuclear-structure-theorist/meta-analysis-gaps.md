## Critical Library Gaps (Priority A)

1. **Richardson-Gaudin exact pairing** (Dukelsky+ RMP 2004): Framework's integrability backbone since S35. N_pair=1 in dim=8 IS the RG regime. Missing this paper means citing without reading.
2. **ATDHFB collective inertia** (Baran+ PRC 2011): Nazarewicz's own paper. Missing it caused M_ATDHFB prediction failure in S40 (predicted 50-170x, actual 1.695).
3. **Ultrasmall grain BCS** (von Delft+ Ann.Phys. 2001): Framework at L/xi_GL=0.031. Mean-field BCS qualitatively wrong here. ED (S35/S37) already correct but missing condensed matter benchmarks.
4. **Pair transfer and GPV** (Potel+ 2014): Framework GPV (B_plus=9.94, 85.5%) needs this for pair transfer form factor computation.
5. **SD band decay-out / chaos-assisted tunneling** (Aberg PRL 1999): B2 dephasing analog. Framework is NON-chaotic (broken analogy) -- need this to understand precisely how.

## Self-Correction Connection

Every Priority A paper addresses a specific past error or gap in reasoning. The library has bulk DFT foundations but misses the exact/finite-size/reaction theory literature most relevant post-S35.

**Why:** The framework shifted from perturbative DFT (S17-S24) to exact ED + compound nucleus (S35-S42). The library hasn't kept up.

**How to apply:** When recommending nuclear analogs or making predictions about the framework's BCS/compound system, check whether the exact solution literature (not just mean-field HFB) supports the claim.
